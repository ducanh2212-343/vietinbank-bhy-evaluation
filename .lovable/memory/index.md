# Project Memory

## Core
App name: "343 Phát triển nhân sự". HR app for banks. Desktop-first, navy/white/gray professional UI. No childish styles.
Skills: 38 total, 4 groups, 4 levels (1-Tân binh, 2-Độc lập, 3-Chuyên gia, 4-Bậc thầy). Core vs Supplementary.
Attitudes: 6 independent groups, not mixed with skills.
IDP limits: Max 3 skills, track all 6 attitudes. Detailed action plans only for weak attitudes.
Assessment flow: 2 steps (Overall vs IDP), stored in separate DB tables.
Positions: 30 positions across 6 depts, each with position_core_skills (min/adv levels).
