# Ảnh thương hiệu "Cây ký ức" — 20 năm VietinBank Bắc Hưng Yên

Upload các file ảnh sau vào đúng thư mục này (`public/brand/`), đúng tên file.
Website tự động dùng ảnh ngay sau khi deploy; khi chưa có ảnh sẽ hiển thị bản vẽ vector dự phòng.

| Tên file (bắt buộc đúng) | Nội dung | Gợi ý |
|---|---|---|
| `cay-ky-uc.png` | Ảnh cây ký ức (cắt từ poster/bộ market) | Vuông ~1000×1000, cây ở giữa; nền trong suốt hoặc nền sáng đều được (web tự bọc khung trắng bo tròn) |
| `huy-hieu-20.png` | Logo/huy hiệu "20 năm 2006–2026" | Nền trong suốt hoặc trắng, ~400×400 |
| `mascot.png` | (Tùy chọn) Linh vật | Nền trong suốt |

Cách upload nhanh trên GitHub: mở thư mục này → **Add file → Upload files** → kéo thả ảnh → Commit.
Nên nén ảnh trước (ví dụ tinypng.com) để trang tải nhanh, mỗi file < 300KB là đẹp.

Vị trí hiển thị: panel trang Đăng nhập, banner trang Tổng quan, dải 20 năm trên sidebar.
